Informations sur le projet

Le concept

Le but de ce site est de visualiser l'évolution des lois sur le droit à l'avortement en Europe. Pour une visualisation simple, nous avons mis en place un code couleur pour que l'utilisateur comprenne en un coup d'œil la situation dans chaque pays en fonction de l'année.

Qui sommes-nous ?

Nous sommes trois étudiantes en B.U.T MMI à Champs-sur-Marne : Estelle FABERT, Emy BIGIRIMANA et Jeanne BESOMBES. C'est dans le cadre d'un projet d'études que nous avons créé ce site nous-mêmes. Pour en savoir plus sur MMI : la formation MMI

Nos sources :

La carte : Leaflet
La majeure partie du code est une version adaptée à ce projet du tutoriel Interactive Choropleth Map du site leafletjs.com de Volodymyr Agafonkin, que vous pouvez trouver ici : https://leafletjs.com/examples/choropleth/.
La base de données Geojson a été quant à elle été tirée de ce site créé par @ashkyd https://geojson-maps.ash.ms/, qui permet de générer des fichiers geojson avec les coordonnées des pays sélectionnés.
Cette base de donnée a été complétée manuellement à l'aide de diverses recherches, notamment sur Wikipédia, que nous avons complétées et croisées avec des articles de presse glanés sur Internet.
